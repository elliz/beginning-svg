What is Jekyll?

  • Simple: database, comments, updates
  • Static: markdown, textile, liquid
  • Blog Aware: permalinks, pages, posts

Getting Started


~ $ gem install jekyll
~ $ jekyll new myblog
~ $ cd myblog
~/myblog $ jekyll serve
# => Now browse to http://localhost:4000

Installation - Mac / Linux


$ gem install jekyll
